Node App

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine.

Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient.

Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Features

  • Fast and scalable
  • Real-time web applications
  • JSON APIs based applications
  • Single Page Applications

Benefits

  • Easy to learn
  • JavaScript everywhere
  • High performance
  • Large community

Installation

Node.js can be installed on Windows, Mac, and Linux.

Download the Node.js source code or a pre-built installer for your platform from the Node.js download page.

After downloading the installer, run the installer and follow the instructions to install Node.js.

After installation, open a terminal and type node -v to check the version of Node.js installed on your system.

Node.js is now installed on your system and ready to use.
